# 🌱 Otimização de Irrigação

> Otimização de irrigação com IA e dados climáticos. Calcula a lâmina ótima por talhão, economizando água e aumentando produtividade.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/otimizacao-irrigacao/

[![Vertical](https://img.shields.io/badge/Vertical-Agro-blue)]() [![IA](https://img.shields.io/badge/IA-ML-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**-31%** Água · **+14%** Produtiv. · **-22%** Energia · **monitor.** Talhões

## 🧠 Tecnologias de IA
ML · IoT · Optimization

## ✨ O que faz
- Lâmina de irrigação ótima
- Dados climáticos INMET
- Sensores de umidade do solo
- Economia de água e energia

## 🗄️ Fonte de dados real
[INMET + sensores solo](https://portal.inmet.gov.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com INMET + sensores solo
2. **Processamento com IA** — ML · IoT · Optimization
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
